# Court rate reference

## 尼崎市記念公園テニスコート

参照先: 公益財団法人尼崎市スポーツ振興事業団「テニスコート」

- 尼崎市内登録団体
  - 平日: 900円 / 面・1時間
  - 日曜・祝日: 1,080円 / 面・1時間
  - 夜間照明: 100円 / 面・30分
- 使用時間: 8:00〜21:00

実装上の前提:

- tennis-app では尼崎市内登録団体料金を使用する。
- 土曜日は平日料金として扱う。
- 祝日は jpholiday で判定する。
- 夜間照明はレッスン時間のうち 19:00〜21:00 と重なる30分単位に加算する。
- 合計額には CoachAvailability.court_count を乗算する。
- 登録画面では自動計算額を初期値とし、実費に合わせた手動上書きを許可する。
